Candy Stripe

How To

  1. Tie all the strings together at the top, leave a 5cm tail for a tie-on when the bracelet is finished.
  2. Clip the knot under the clipboard.
  3. You will now have about 60cm of each string hanging down ready to knot. Put them in the order you want the stripes to appear, left to right.
  4. Left to right, the strings will be 1, 2, 3, and 4.
  5. Take string 1, cross it over string 2 to make a forward knot (don't forget to do the shape twice to make one forward knot.)
  6. Now take string 1 and cross it over string 3 to make a forward knot.
  7. Now take string 1 and cross it over string 4 to make a forward knot.
  8. You will now have a new leader string. String 2, will now be the leader.
  9. Using string 2 as leader. Do the same across 3, 4 and 1.
  10. String 3 will then be the leader.
  11. Keep going across like that until your bracelet is long enough. And there you have it! One candy stripe friendship bracelet.
